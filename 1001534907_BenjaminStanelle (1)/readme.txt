Running the server and client:
Download the server.py and put it in a folder along with test2.html and any other server side objects.
Put the client.py in another folder or the same folder.
Navigate to server.py using command prompt and run in the command line using the command "python server.py"
In another command prompt window navigate to the client.py and run it using the command "python client.py {ip address} {port number} {file name}"
If any of the fields following python client.py are left blank it will use the default values 



default file name: test2.html
default port: 8080
default ip: (automatically connects to socket server over port 8080 on the local network)
